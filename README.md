# api-game-finder
Aplikasi web sederhana untuk mencari game menggunakan REST API.

# Fitur
- Landing page (cover)
- Search game berdasarkan nama
- Menampilkan rating & release date

  # Teknologi
- Python
- HTML & CSS
- REST API (RAWG)

# Cara Menjalankan

1. Install dependency:
pip install flask requests

2. Jalankan aplikasi:
python app.py

3. Buka di browser:
http://127.0.0.1:5000
